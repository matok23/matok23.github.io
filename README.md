# Portfolio

Personal portfolio website built with React, TypeScript, and Vite.

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- CSS

## Features

- Responsive design
- About/profile page
- Projects showcase
- Downloadable CV (English & Slovak)
- GitHub Pages deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
